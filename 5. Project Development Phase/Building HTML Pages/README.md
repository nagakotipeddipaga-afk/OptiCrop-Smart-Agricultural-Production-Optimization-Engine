This section shows the navigation structure of the OptiCrop web application homepage. The HTML page contains a top navigation menu with links to Home, About, and FindYourCrop pages. The homepage introduces the purpose of the Smart Agricultural Production Optimization Engine and provides smooth navigation between different application modules.

This page presents detailed information about the OptiCrop project and explains the objectives of intelligent crop prediction using environmental and soil parameters. The About page describes how machine learning techniques are used to improve agricultural productivity, optimize farming decisions, and support sustainable agricultural practices.

This section displays the crop prediction form where users provide agricultural parameters such as Nitrogen, Phosphorous, Potassium, temperature, humidity, pH, and rainfall values. The HTML form collects user input and sends the data to the backend prediction system for intelligent crop recommendation generation.
